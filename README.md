# Unemployment-Analysis-Multilinear-Regression-STA302
</br>
Class Project on multiple linear regression. </br>
A multiple linear regression (MLR) is conducted on the Canadian economy using quarterly economic data. The relationship between the unemployment rate and key economic indicators (GDP growth, Prime Rates, Inflation, Immigration, and Investment Rates) is investigated. R code used for analysis (exploratory data analysis (EDA), selection of model, final validation of the model) were combined into one single R file. The final report and raw data are also provided to provide necessary background information. 

</br>
All economic data comes from the OECD database: https://data.oecd.org </br>
Yearly immigration statistics come from Statistics Canada: https://doi.org/10.25318/1710004001-eng
